What is Auto Super Grub Disk ?

Auto Super Grub Disk is an special Super Grub Disk fork. It can fix the Linux boot without having to ask anything to the user who do not have to bother about what option should (s)he choose.

Where I can get / download it ?

Currently you can download it from SGD forjamari downloads.

How do I use it?

The CAPITAL_LETTERS_WORDS will be substitued with screenshots in a future.

Once you have downloaded the file double-click it.

WINDOWS_DESKTOP_WITH_FILE_SELECTED

A wizard will guide you towards installing unetbootin-supergrubdisk into your system.

WIZARD_FIRST_STEP

Once the wizard has finished you will be prompted to reboot.

SEVERAL_WIZARD_STEPS

WIZARD_ASKS_YOU_TO_REBOOT_STEP

Reboot (just after saving your open important documents). Now you should see the Windows boot menu with another option: unetbootin-supergrubdisk.

WINDOWS_BOOT_MENU_WITH_UNETBOOTIN-SUPERGRUBDISK_OPTION_UNSELECTED

Select it with the down arrow and press enter when it is hightlighted. Now do nothing.

WINDOWS_BOOT_MENU_WITH_UNETBOOTIN-SUPERGRUBDISK_OPTION_SELECTED

If STEP 1 (Fix of boot) goes ok you will see an smile countdown screen, still do nothing.

AUTO_SGD_FIX_OF_BOOT_SMILE

If STEP 2 (Boot of Linux) goes ok you will see your grub menu again.

GRUB_MENU

Select your Linux and boot it.

LINUX_DESKTOP

Once you are in your Linux reboot

GNOME -> SHUTDOWN SHORTCUT

and you will see that you see Grub menu instead of going directly to Windows.

GRUB_MENU

Once booted in Windows the wizard will ask you if you want to remove Super Grub Disk from the windows boot menu, answer yes.

WIZARD_ASKS_REMOVE_UNETBOOTIN_SUPERGRUBDISK_BOOT

(Do not be bothered about this because you will still have the downloaded auto_super_grub_disk_1.0.exe file intact.)

WINDOWS_DESKTOP_WITH_FILE_SELECTED

Problems ?

If you have any problems or Auto Super Grub Disk does not seem to make its work you can check the Auto Super Grub Disk problems page
